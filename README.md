# landing-page
Landing-page is a simple and non-responsive website which has a hero
part that represents the deader, content which has been divided 3
parts namely the info-header which uses the title, the info-container
which also houses the information and their associated images,
the quote which contains the quote or the message shared on the website
by God of Thunder and the fourth being the container which keeps the 
call to action and it's associated button.
The footer section also houses The Odin Project copyright inscription.

# About
This project is an assignment which was embarked on by
Kwamed2k (a student of The Odin Project) and it seeks to introduce
the author of this to the basic knowledge of HTML and CSS (flex-box).

HTML and CSS (flex-box)
